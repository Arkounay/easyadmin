Documentation
-------------

#### Installation

  ```bash
  $ git clone git@github.com:artgris/easyadmin.git
  $ rm -rf .git && git init && git add . && git commit -m "First commit"
  $ git remote add origin git@github.com:...
  $ git push -u origin master
  $ cd easyadmin
  $ composer install
  ```
